**Guide d’installation Orcaslicer et profile des imprimantes 3D Wemake3D**

1.  **Installation de l’OrcaSlicer :**

OrcaSlicer est le logiciel de préparation à l’impression 3D. C’est l’outil de création du G-code.


Lien de téléchargement : [Orca Slicer » 3D Printing Slicer \[Download Now\]](https://orca-slicer.com/#download)

Suivez les étapes jusqu’à la fin

1.  **Installation du profil :**

Naviguez vers le dossier d’installation du logiciel Orcaslicer

Normalement c’est dans `C:\Program Files\OrcaSlicer\resources\profiles`

Et copier les fichiers téléchargés depuis ce lien :

[https://github.com/WemakeTN/OrcaSlicerProfiles/releases/tag/WEMAKE3D\_Patch\_V0.1](https://github.com/WemakeTN/OrcaSlicerProfiles/releases/tag/WEMAKE3D_Patch_V0.1)

Nom des fichiers :

*   `WEMAKE3D (dossier)`
*   ``WEMAKE3D.Json (fichier Json)``

1.  **L’ajout de l’imprimante dans Orcaslicer :**

Suivez les étapes jusqu’à cette fenêtre :
